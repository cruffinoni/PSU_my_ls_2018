readdir:
    - ionde number
    - folder name (256 caractères)
    - type (d_type)
    - offset to the next directory
RETURN VALUE:
    On success, readdir() returns a pointer to a dirent structure

strerror:
    - Permet de récupérer le message d'erreur relié à errno et le stocker
        dans une chaîne de caractères

perror:
    #include <stdio.h>
    void perror(const char *s);
   The perror() function produces a message on standard error describing
   the last error encountered during a call to a system or library
   function.

time:
    - Retourne le nombre de secondes depuis 1970 AKA UNIX TIMESTAMP sous forme
    d'une structure -> À utiliser avec ctime !

ctime:
    - Renvoie une chaîne de caractères ou NULL si erreur.
    (#include <time.h>)
    Prends en paramètre: const time_t *time

getpwuid (struct passwd *getpwuid(uid_t uid)):
    #include <sys/types.h>
    #include <pwd.h>
    - Retourne un pointeur vers une structure

getgrgid:
    Permet d'avoir des informations concernant un groupe. Retourne un pointeur
    ou NULL si erreur

stat:
    stat() stats the file pointed to by path and fills in buf.
    lstat() is identical to stat(), except that if path is a symbolic link,
    then the link itself is stat-ed, not the file that it refers to.
    Retourne -1 en cas d'erreur et errno est set.

readlink:
    Permet de lire un lien symbolique à la différence de readdir qui ne peut pas.
    À vérifier..
